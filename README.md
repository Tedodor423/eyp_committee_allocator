# EYP committee allocator
alokace komisí

## Instalace
nainstalujte nejnovější [Python](https://www.python.org/downloads/)

stáhněte si kód zde: [committee_allocator-orderscaleV2.py](https://github.com/Tedodor423/eyp_committee_allocator/releases/download/release/committee_allocator-orderscaleV2.py)

## Použití

stáhněte tabulku s informacemi o delegátech v .xlsx
přesuňte `committee_allocator-orderscaleV2.py` do stejné složky a spusťte jej

## Soubory

`/src/committee_allocator-orderscaleV2.py` - hlavní soubor, v komentářích popsaná funkce
`/src/deprecated/*` - jiné algoritmy které nefungují nebo fungují hůř
